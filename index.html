Introduction to Programming

The internet is probably one of the most fascinating things ever created. One can find anything at anytime and even more than that the internet connects people from anywhere in the world.

The Internet has two faces: the one that we see, with beautiful made and websites and easy to work with and the one that we don't see. The face that we don't see is a bunch of written text that is called HTML language. So everything on the internet is made up of documents written in HTML.


HTML, known as Hypertext Markup Language is the language that computers understand. And all the HTML documents are connected on the internet thruough Hyperlinks.


TAGS AND ELEMENTS

HTML language contains elements that are each defined by tags. When we write HTML code, tags help us tell computers what each element represents.


In HTML language, elements can be either Inline or Block.
The inline elements appear in a straight line and the block elements have an invisible block around them.
